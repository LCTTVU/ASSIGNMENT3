Add a phone

POST request to /phones

Type: JSON

Sample Post:

{

"image": "https://media.s-bol.com/qZLzw6x7rA1D/550x747.jpg",

"brand": "Samsung",

"model": "Galaxy S21",

"os": "Andriod",

"screensize": 6

}

Sample Response:

{

"id": 3

}
