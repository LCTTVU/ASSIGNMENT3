API documentation

The server can be started by using the command "npm start" in the terminal.

The server will be running at http://localhost:3000.

Users can interact with the server and database using the requests below.

Get ALL phones

GET request to http://localhost:3000/phones

Sample Response to request: GET http://localhost:3000/phones

    {
        "rows": [
            {
                "id": 1,
                "image": "https://static.iphoned.nl/orca/products/11756/apple-iphone-x.png",
                "brand": "Apple",
                "model": "Iphone X",
                "os": "IOS",
                "screensize": 6
            },
            {
                "id": 2,
                "image": "https://media.s-bol.com/qZLzw6x7rA1D/550x747.jpg",
                "brand": "Samsung",
                "model": "Galaxy S21",
                "os": "Andriod",
                "screensize": 6
            }
        ]
    }

In case of success: All items' info can be retrieved, API will return 200 OK

In case database not found: API will return 404 Not Found

In case of other errors: API will return 400 error:+error message

Get specific phone

GET request to http://localhost:3000/phones/phoneID

Sample Response to request: GET http://localhost:3000/phones/1

    {
        "row": [

                "model": "ssssssssssss",
                "os": "ssssssssssssssss",
                "screensize": 26
            }
        ]
    }

In case of success: An item's information can be retrieved, API will return 200 OK

In case phone not found: API will return 404 Not Found

In case of other errors: API will return 400 error:+error message

Add a phone

POST request to http://localhost:3000/phones

Input type: JSON

Sample POST Request to: http://localhost:3000/phones/

    {
        "image": "https://media.s-bol.com/qZLzw6x7rA1D/550x747.jpg",
        "brand": "Samsung",
        "model": "Galaxy S21",
        "os": "Andriod",
        "screensize": 6
    }

In case of success: A new item will be created, API will return 201 Created, with the following sample response:

    {
        "id": 3
    }

In case of other errors: API will return 400 error:+error message

Update a phone

PUT request to http://localhost:3000/phones

Input type: JSON

Sample PUT Request to: http://localhost:3000/phones/2

    {
        "id": 2
        "image": "https://media.s-bol.com/qZLzw6x7rA1D/550x747.jpg",
        "brand": "Samsung",
        "model": "Galaxy S21",
        "os": "Andriod",
        "screensize": 16
    }

In case of success: The item's info will be updated, API will return 200 OK, with the following sample response:

    {
        "updated": 1
    }

In case the item is not found: API will return 404 Not Found

In case of other errors: API will return 400 error:+error message

Delete a phone

DELETE request to http://localhost:3000/phones/phoneID

Sample DELETE Request to: http://localhost:3000/phones/2

        {
            "deleted": 1
        }

In case of success: The item with the matching ID will be deleted, API will return 200 OK

In case the item is not found: API will return 404 Not Found

In case of other errors: API will return 400 error:+error message
